# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-21

### Added
- 🎉 Initial open source release
- 📊 Real-time dashboard with agent metrics
- 🎮 Gamification system (XP, levels, achievements)
- 📈 Five-dimension scoring (efficiency, output, automation, collaboration, accumulation)
- 🏥 Four-status monitoring (energy, health, mood, hunger)
- 📉 7-day growth history charts
- 🌐 Multi-agent overview and ranking
- 🤝 Collaboration network visualization
- 🛠️ Tool usage analytics
- ⏰ Cron job monitoring
- 📁 Workspace analysis
- 🔐 Password authentication
- 🌍 Bilingual support (Chinese/English)
- 🐳 Docker deployment support
- 📱 Responsive dark theme UI

### Technical
- FastAPI backend with SQLite storage
- Built-in scheduler for hourly collection and daily cleanup
- Incremental data collection to prevent database bloat
- Environment variable configuration
- RESTful API design
